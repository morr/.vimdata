html_escape(s)

A utility method for escaping HTML tag characters in s. 

  require "erb"
  include ERB::Util

  puts html_escape("is a > 0 & a < 10?")

Generates 

  is a &gt; 0 &amp; a &lt; 10?

# File erb.rb, line 755
    def html_escape(s)
      s.to_s.gsub(/&/, "&amp;").gsub(/\"/, "&quot;").gsub(/>/, "&gt;").gsub(/</, "&lt;")
    end
