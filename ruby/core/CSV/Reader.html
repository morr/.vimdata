Class CSV::Reader
In: lib/csv.rb
Parent: Object

CSV formatted string/stream reader.

EXAMPLE 

  read CSV lines untill the first column is 'stop'.

  CSV::Reader.parse(File.open('bigdata', 'rb')) do |row|
    p row
    break if !row[0].is_null && row[0].data == 'stop'
  end
